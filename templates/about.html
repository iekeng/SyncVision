The Technologies

The project Syncvison is a web application that connects user on the internet, enabling real time communication between them. The application Implements this real time communication, using web technologies:

  • webSockets
  • webRTC

The above technologies enables the core of the application; real time communication, while the following technologies handled the other parts of the application, bringing it to life:

    Other technologies used:
  • Python Flask
  • Python FlaskSocketIO
  • HTML
  • CSS
  • JavaScript
